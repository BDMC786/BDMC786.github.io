Summary: vs. X

This analysis looks at the ways in which the weather changes with proximity to the equator. Weather data for over 500 cities was analyzed for temperature (pictured left), cloudiness, wind speed, and humidity. These results were plotted to determine if any of these aspects of weather are influenced by latitude.

Available on this site is an analysis of the findings for each of these factors, including a plot of the data, as well as a page containing all of the plots together for comparison purposes. Also included is the raw data table obtained for each city included.

Data was obtained by generating a random set of latitude and longitude coordinates using the random library in Python, then using the cities library to generate a list of cities based on these coordinates. Weather data was obtained by searching these cities from OpenWeatherMapAPI. Results were plotted using MatPlotLib.
